Java中的继承、重载与重写

发表于 2019-09-12   |   分类于 编程语言

继承

  • 子类通过关键字extends关键字继承父类
  • 子类可以调用父类中的属性与方法（public、protected）
  • 子类可以添加自己的独有成员
  • 子类继承了父类的方法，但不能满足子类需要，方法可以重写

重写

  • 子类修饰符权限可以大于等于父类
  • 父类用final修饰，子类不能重写；父类用static修饰，子类不存在重写；父类用abstract修饰，子类必须重写
  • 子类返回值类型小于等于父类
  • 子类方法名与父类一致

重载

  • 方法名字相同，参数列表不同
  • 参数不同体现在类型、个数、顺序不同
  • 返回值类型可随意，不能以返回类型作为重载函数的区分标准
